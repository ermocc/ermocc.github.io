提示

本文主要讲解如何使用 VuePress 快速搭建个人博客网站。

我的环境版本号。

# 1.22.10
yarn -v

# v14.15.1
node -v

# 6.14.8
npm -v

创建并进入一个目录。

mkdir ermo && cd ermo

初始化包管理器。

yarn init

将 VuePress 安装为本地依赖。

yarn add -D vuepress

创建第一篇文章。

mkdir docs && echo '# Hello VuePress' > docs/README.md

在 package.json 中添加启动脚本。

{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

本地启动服务器。

yarn docs:dev

访问 http://localhost:8080 就可以看到一个简单页面，当前服务器支持热部署，修改代码页面会实时刷新。

此时的项目目录结构是这样的。

|-- ermo
    |-- package-lock.json
    |-- package.json
    |-- yarn.lock
    |-- docs
        |-- README.md
