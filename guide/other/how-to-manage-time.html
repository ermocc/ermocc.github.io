# 为什么精英都是时间控

桦泽紫苑先生每年要出版3本书，每天更新网络日志、YouTube、Facebook、博客。每月读书20册以上，并发表书评。每月举办2~3次研讨会，每年去海外旅行30天以上，这样的日子在写这本书的时候已经延续了7年。

# 为什么会看这本书

作为一个上班族，每天加完班后我还想回家学习一会。我一般学习到12点，然后睡觉。最近我就觉得很痛苦，上本书埃里克 · 杰克逊的《支付战争》让我整整看了一个星期，而且每天晚上我的注意力不集中，并且特别想睡觉。我一回家洗漱完就开始看书，到家了也没有和女朋友沟通的时间。这时候我就在想，上班族肯定也可以用业余的时间来充实自己，但我的做法显然是不对的。于是我就偶然找到了这本书《为什么精英都是时间控》，看上去又像是一本心灵鸡汤、纸上谈兵的书。事实却不是这样的，如果你也有像我一样的困扰，那么我强烈推荐你来读一下这本书，一定会受益匪浅。

# 如何提高工作与学习效率

书中写到，从早晨到上午的这段时间，是头脑机能最好的时间段。这段时间被称为大脑的黄金时间，特别适合做理论性强的工作，比如创作理论性文章、学习外语以及需要高度专注的工作。并且上午的时间的价值是晚上的4倍。我开始读这本书的时候就是早上读的，并且用了一天的时间就读完了这本书，也算是体会到了作者说的话。原来我工作以后的学习时间段选错了。至少我本人绝对是早上的精力比晚上要充沛不少。

除了选择自主学习的时间段以外。学习的场所也很重要，作者研究过，当我们来到陌生的地方时，场所神经元就会异常活跃，场所神经元也可以增加我们的注意力和积极性。所以在中午吃饭的时候一定要出去吃，在太阳下散步一段距离。

下午2点到4点的时候人最容易犯困，这个时候的工作效率是最低的，运动可以让人保持活力，如果你像我一样在办公室的时候，爬楼梯和深蹲是一个不错的选择。

给自己设定下班时间，作者在美国留学的时候，发现美国人都是5点就下班，他们下班后陪自己的家人吃饭。这是一个很好的习惯。为了能在5点下班，所以他们愿意在工作时间提高效率，这在无形之中给自己设定了一个截止日期。作者平时也是这样，新的一天开始时，他就会在互联网上给自己订一张晚上6点电影票。这样，他就会在白天的时间内容更加认真的写作，才不会错过晚上的电影。我觉得这个主意非常棒，我平时也会遇到这样的场景，明天晚上一个老同学要过来，所以我在白天就会特别认真的工作，争取早点把工作做完，这样晚上就可以和同学去小酌一杯了。如果我们每天可以主动地给自己设定一个截止日期，那工作效率肯定可以提高。

# 15 · 45 · 90法则

为什么电影《007》的动作戏每过15分钟就会出现邦德和邦德女友的缠绵，这和整个剧情毫无关系。为什么学生们的上课时间是45分钟？为什么一场足球赛的时间是90分钟？因为我们的大脑就像一根弦一样，不可能绷的太久。在电影《007》中，15分钟的动作戏让观众的大脑处于高度集中的状态，15 分钟后的缠绵戏正好是观众的放松大脑的时候。学生们上课的时候也是这样，学生们的上课时间只能持续45分钟左右，其实很多老师在上课的时候都会将一节课45分钟分为3个15分钟的片段。我们可以将同样的道理放到工作中，每过45分钟就休息一下 ，适当的放松可以让我们进入下一轮的高度集中状态。

# 蔡戈尼克效应

苏联心理学家蔡格尼克在一家他经常光顾的咖啡馆中有一个重大的发现。咖啡馆的服务员不用做记录就能同时、准确地记住好几位客人的点餐记录。可是当这位服务员把客人的餐点端上桌后，他很快就把刚才那几位客人点了什么都给忘记了。

“当目标没有实现时，人对未完成的课题记忆比较深刻。但目标实现后，人对已经完成的课题，记忆就没那么深刻了。”人们将这种现象称为“蔡戈尼克效应”。

电视剧中每一集快要结束的地方都是设置很多悬念。让人们很期待下一集的播放，这就是利用了蔡格尼克效应。我们在工作中也会有很多任务，如果这些任务没有完成的话，他们就会一直存在我们的脑海中，这样会分散我们的注意力。解决办法很简单，将需要做的任务全部记录笔记本上，做一个 TODO 清单，放到桌子上，这样我们在当下的工作时间中就会全神贯注。

# 结尾

书中还有很多好的想法，用来提高工作效率，以下的语句都来自书中。

在干净、整洁的办公桌前效率更高。

整理物品最重要的一点就是每种物品都有自己固定的位置。

专注中闪现的1秒钟的杂念，都会浪费我们15分钟重新集中注意力。

将待办事情用笔写下来，写成 TODO 清单，清空大脑。

我终于意识到“写文章，必须要在身体和头脑还没有疲惫，注意力最为集中的上午进行”。

在进行有氧运动的时候，头脑会分泌一种名叫BDNF（brain derived neurotrophic factor，脑源性神经营养因子）的物质，BDNF对脑神经的成长发育和正常运转起着至关重要的作用。此外，头脑还会分泌一种叫作多巴胺的神经递质，多巴胺能够提高人的兴致，使人产生幸福感。

我认为，节省出来的“自由时间”应该用来进行“自我投资”“主动性娱乐”和“享受人生”。

中午进行26分钟的小睡，可以让下午的工作效率提高34%、专注力提高54%。

听音乐、闻香薰精油、吃美食、泡个热水澡、蒸桑拿、按摩、听河水流淌的声音、晒太阳、爱抚宠物……我想上述方法可能是朋友们喜爱的放松方式。可是您发现没有，这些方法都有一个共同的特点，就是五感之中，都没有用到“视觉”。

闭目养神的一个要点是头脑要放空，什么都不想。从脑科学的角度来说，这就是最为简单易行的休息术。

换一种更为形象的说法就是，您每运动15分钟，就可以多活8个小时！听起来还是非常诱人的。

不让别人等待，按照预定时间办事，是对他人时间的最大尊重。

首先要了解自己最快乐的瞬间是什么时候，然后不断增加这样的瞬间，人生就会越来越幸福！
