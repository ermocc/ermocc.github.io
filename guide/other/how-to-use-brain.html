# 为什么精英这样用脑不会累

读完桦泽紫苑先生的上本《为什么精英都是时间控》收获非常大，我调整了业余的学习时间段，并且实践了一些小技巧，用来提升工作效率。既然时间是可以科学利用起来的，那我们人类的核心，大脑也是可以科学的进行输入输出知识。所以我决定再来学习下作者的另一本书《为什么精英这样用脑不会累》。作者桦泽紫苑先生是一位神经科医生，本书主要从医学的角度来讲解大脑的工作机制，以及各种情绪的产生。然后我们如何来自主的控制这些情绪。

# 多巴胺=幸福

比利时作家的童话故事《青鸟》，兄妹蒂蒂尔和米蒂尔二人在梦中去了过去和未来之国去寻找象征着幸福的“青鸟”，经历了一段奇妙的旅行后，他们并没有找到“青鸟”。后来他们回家后，发现自己家中的鸽子正是那只象征着幸福的“青鸟”。这个故事告诉我们其实幸福就在我们的身边，只是我们还没有发现而已。我们中国也有句古话，说的就是这种现象，叫身在福中不知福。

其实“幸福”就在我们的大脑中，准确的说，当我们的大脑在分泌多巴胺的时候，我们就能体验到幸福的感觉。那么怎样才让大脑分泌多巴胺呢？当人在制定目标、目标实现、被表扬的时候都会产生多巴胺。作者研究出了促进多巴胺分泌的七个步骤。

  1. 设定明确的目标，只有在人们设定切实可行的目标后，并且不断反复思考，努力实现的过程中，才会分泌多巴胺。
  2. 想象自己实现目标后的样子，其实就是心理暗示，心理暗示会提高成功的概率。
  3. 对目标进行反复确认，将目标写到纸上，放到身边，不断的暗示自己，也可以补充多巴胺的分泌。
  4. 快乐的朝着目标努力
  5. 实现目标后要给自己充分的奖赏，人脑很“贪欲”，获得奖赏后还想再次获得奖赏，这种“贪欲”也会促进多巴胺的分泌。
  6. 一个目标达成后，立即设定更高的目标
  7. 不断重复实现目标的过程

# 去甲肾上腺素：“恐惧”和“压力”也可以提高工作效率

日本著名的礼仪讲师平名都女士，每年要举办至少300场礼仪讲座。平名都在举办讲座的过程中，都会满面笑容，以优雅和蔼的方式开场。但是如果要有学生把问题回答错误的话，她马上就会提起嗓子怒斥学生，学生马上就会注意力高度集中，专心听讲。正是因为这种严厉的授课方式，讲座的效果特别好，学生们都说收货非常大。这种场景在我的学生时代也是非常常见的，我读到这里，大脑中就会不自觉的浮出一幅画面，下午3点左右，语文老师老师在讲台讲文言文，我的眼皮不由自主的往下掉，忽然传来班主任的声音：“xx，你给我站起来，我在讲课你听没听，要睡回家睡去。”被这么训斥一下，我这一下午都不会再打盹了。

这就是去甲肾上腺素在作怪。古时候，我们的祖先每天都要面临着生存的巨大考验。比如说一个人类在一只凶残的剑齿虎面前，根本容不得人类去迟疑，他要迅速做出逃跑还是战斗的选择，这个时候就是在分泌去甲肾上腺素，这种逃跑或战斗激素可以在短时间内提高人的专注力，同时还有“镇痛剂”的作用。

# 肾上腺素：把“愤怒”和“兴奋”变成朋友

当肾上腺素被释放到血液中后，人的心跳速度会加快，血压会上升，以便将血液输送到肌肉中。同时，血糖升高、瞳孔扩大，头脑变得更加清醒，专注力提高，让身体和大脑做好“临战准备”。

这么一听，去甲肾上腺素和肾上腺素的功能差不多。其实二者作用的主要部位不同，去甲肾上腺素主要作用于脑和神经，而肾上腺素主要作用于器官和肌肉。我们在电视上或互联网上经常可以看到这样的画面，一场拳击赛开始前，比赛选手会用面部表情和眼神嘲讽、鄙视对方。其实这就是在有意的促使肾上腺素的分泌，从而在赛前达到最佳的状态。

肾上腺素不是分泌越多越好的，肾上腺素分泌过剩会给人带来过度的压力。有的人在早晚高峰挤地铁的时候就会有很大的精神压力。这就是肾上腺素分泌过剩的体现。作者在书中提出了环境的压力的深呼吸法。

  1. 首先，正直站立。
  2. 有意识的将身体重心放在肚脐以下15厘米的地方，收紧肛门括约肌。
  3. 用鼻子吸气，再用鼻子呼气。
  4. 深吸气5秒，屏住呼吸一小会儿，再深呼气7秒。

# 血清素：克服睡懒觉得靠它

我们可以想象F1赛车在绿灯亮起前，所有车辆的引擎都已经开始嗡嗡作响，绿灯亮起时，赛车手们立即踩下油门，所有赛车以最快的速度冲了出去。如果绿灯亮起前没有进行预热，直接进行加油，就会损坏发动机，或者熄火。血清素在我们的大脑中就起到了预热的作用。早晨在光的刺激下，大脑开始分泌血清素，在血清素的指导下，我们的身体状态从休息调节到苏醒。所以怎样科学的起床就变得非常重要了。

桦泽紫苑先生建议，晚上开着窗帘睡觉，或者窗帘的厚度要薄一点。这样早晨的太阳升起时，我们的身体受到阳光的刺激，大脑就会分泌血清素，血清素开始调节身体的机能，从而克服赖床的习惯。起床后可以出去散步30分钟左右，这样可以将身体的状态调整至最佳。

感动的眼泪也可以提高血清素的活力，作者非常推荐通过看电影来磨炼共感力，从而提高血清素的活力。好的电影都细节刻画非常细腻，可以轻松的带动观众的情绪。读到这里，我想到了一部印度电影《摔跤吧，爸爸》，我确实在看的时候留下的了感动的眼泪。

# 褪黑激素：“睡眠物质”将身心彻底修复

褪黑激素是人类于1958年首次发现的。它不仅仅会对脑神经产生作用，还能使心跳减慢，体温、血压下降，并调整人入睡与醒来的规律，具有促进自然入睡的作用。褪黑激素可以将我们全身的各种器官切换到休息模式。所以，人们也把褪黑激素称为“睡眠物质”“催眠激素”。

好的睡眠可以让我们第二天一整天元气满满，有的人经常会失眠，这就是因为褪黑激素分泌过少的原因。作者建议在睡前关闭卧室强光灯，并且打开床头灯，听会音乐，看会书，当睡意来临的时候，放下书本自然入睡，这是最好的入睡方式。

# 乙酰胆碱：灵感的来源

一天晚上，俄国化学家门捷列夫在打完牌后一个人在思考元素之间的各种关系，当睡意来临的时候，他就睡过去了。当他在此睁开眼睛的时候，忽然悟出了元素之间的关系，然后马上编写出了闻名世界的元素周期表。

我们的梦之所以荒诞离奇、天马星空，就是因为胺类物质的理性束缚变得最弱，乙酰胆碱激发了头脑的想象力、创造力。再一步分析，灵感闪现还与记忆有关。

因此我们要想提升灵感闪现的概率，我们需要往大脑中输入大量的信息。如果读很多书、接触很多人、反复经历很多事情、尝试体验失败，大脑中的信息越多，这些信息相互碰撞出的火花（灵感）就会越大。

外出散步、午睡、坐着活动手脚、刺激好奇心等，都可以容易闪现出奇妙的灵感。另外，长期抽烟的话，身体就不再自己合成乙酰胆碱了。

# 第七章 内啡肽：脑内毒品

在受到压力刺激的时候，我们脑中的下垂体开始分泌内啡肽。内啡肽与分布于大脑皮质、丘脑、脊髓等处的“阿片受体”结合，除了镇痛作用之外，还有减少胃肠蠕动、缩瞳、产生幸福感、减缓脉搏、抑制神经递质等作用。

有些毒品，比如吗啡，海洛因等就是和阿片受体结合，人类才会体会到毒品带来的“恍惚感”，有就是飘飘欲仙的感觉，从而形成毒品上瘾。

内啡肽可以带来两种效果，一种是“轻松治愈的效果”。以下活动会带来这种效果。

  • 听古典音乐的时候
  • 听自己喜欢的音乐的时候
  • 听潺潺流水声的时候
  • 看大海、红叶等美景的时候
  • 吃好吃的食物时·清风拂面的时候
  • 闻到薰衣草等植物香气的时候
  • 闭上眼睛安静放松的时候
  • 集中注意力做一件事的时候
  • 心情平静的时候
  • 练习瑜伽、冥想、坐禅的时候

另一种就是“快感刺激的效果”，当只有多巴胺分泌的时候，人的幸福感是1倍，当脑内同时分泌多巴胺和内啡肽的时候，人的幸福感就会提升到10~20倍。以下活动会带来这种效果。

  • 运动
  • 吃辛辣的食物
  • 吃油腻的食物
  • 吃巧克力
  • 泡热水澡
  • 针灸治疗

如果我们可以灵活的控制并且平衡这两种效果，那我们的生活状态一定会变得轻松、无压力。
