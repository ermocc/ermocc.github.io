# 阅读为什么这么难

好好学习，天天向上。怎样学习才是好好学习？你有没有仔细思考过这个问题。学习不外乎就是为了获取有效的信息，目前学习无非这四种方式：拜师，互联网获取信息，自己不断的踩坑，阅读。也称为人、网、事、书。

这四种方法中，如果想系统性的学习一项知识，只有拜师和阅读这两种方法。拜师的门槛比较高，俗话说千里马常有，伯乐不常有。如果你的生活和工作中有这么一位德高望重的长者为你指路，一定要好好珍惜。

除了拜师，想要系统性的学习一项知识，就只剩阅读这种方法了。但是阅读对我们有益，我们都知道。但是大家普遍的想法都是：嗯，我知道阅读很好，但是我想去打两把王者。

为什么？为什么我们明明知道阅读对我们有好处，但我们就不想去做呢？有没有什么办法可以让我们慢慢的喜欢上阅读呢？今天我们就来讲讲作者彭小六的《洋葱阅读法》。

# 学习前先了解我们的大脑

众所周知，我们都是从猴子进化过来的。所以大脑也经历了这样三个阶段：爬虫脑、哺乳动物脑、皮质脑。

爬虫脑时期我觉得就是躺平脑，怎么舒服怎么来。

哺乳动物脑时期，就和现在我们见到狮子、老虎等动物的反应是一样的，比如说最近的云南大象迁徙的事情。大象们会为了食物起冲突，它们是有情绪的。

人类经过几千数万年的进化，现在拥有了皮质脑。这个时期的脑子让我们拥有了理性，该生气的时候我们的理性会告诉自己：冲突是魔鬼，生气解决不了问题。当我们有三急的时候，我们不会随地解决，我们会去卫生间。这就是皮质脑在起作用。

了解了我们的大脑后，这时候你再想想，如果突然给你拿过来一本书，你的大脑会怎么想。爬虫脑：让我学习？没门。哺乳动物脑：书是干什么的，这书会不会对我的生命有危险，我不感兴趣。所以，看到我们不感兴趣的东西，我们的大脑会本能的抗拒，这不是你一个人的问题，所有人类都是这样。

**那有没有什么办法能让我们对阅读感兴趣？**答案是有的。

首先，我们要带着问题，带着目标去阅读。阅读之前，先看下封面、序言、目录。目录是一本书的主要框架。在阅读之前根据目录提出问题，然后带着问题去阅读。

其次，联想很重要。我们的大脑其实就是一个关系型数据库，孤立的知识点很容易忘记。每次看到一个新的知识点的时候，一定要和自己学过的知识进行关联，建立关系。

比如说「比特币」这个概念。第一次看到的人可能会有点懵，但是有人要是告诉你，比特币就是互联网上的一种钱币，相当于一种虚拟的货币。听到这里你可能就对比特币有个大概的概念了。因为你将「比特币」和已有知识建立了联系，互联网你是知道的，钱币你也是知道的。

最后，分享是一个反复学习的过程。

有一句话是这样说的。

老师是在教学生的时候才真正理解这个知识点的含义。

在书本中阅读到新的知识后，这个时候知识在我们的大脑中属于租户（短期记忆），只有通过不断的理解与转化才可以加深记忆线索，最终让这些知识在我们的大脑中拿到身份证（永久记忆）。

怎样分享？写文章、讲课都可以，前提是要用自己的理解方式向别人表述。

# 阅读环境总是很吵怎么办？

番茄钟是个好东西，有一本书就叫做《番茄工作法图解》。这本书的核心就是介绍了一种学习方法来提高工作和学习效率，那就是番茄钟。

什么是番茄钟呢，就是你自己定一个25分钟的闹钟，在这个时间段内，你只做一件事，那就是阅读。期间如果自己思绪被打乱的话，那就将注意力集中到课本上。25分钟过后，就休息5分钟，如此循环。

人的高度集中时间是有限的，你仔细想想，生活中有很多这样的例子。电影每次在精彩的动作戏后会加一段无关紧要的剧情。学生的上课时间一般都是45分钟。所以番茄钟可以将我们集中精力的时间透明化，番茄钟就像在监督你一样，强行让你认真做一件事。

# 什么时候读书比较靠谱呢？

解答什么时候读书前先了解另一个概念，精力电池。就是字面意思，人一天的精力就像一块电池，白天开始各种浪（耗电），晚上进入睡眠状态（充电）。每个人的体质不一样，什么时候精力旺盛只有你自己最清楚，大部分人的精力是早上最好，因此只要在你电量最足的情况下进行阅读，肯定事半功倍。
