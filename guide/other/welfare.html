# 我问你幸福吗，你却告诉我你姓李？

每个人都有梦想，没有梦想那和咸鱼有什么分别。

# 幸福像爬山一样有终点吗

泰勒也有梦想，他的梦想是成为壁球比赛的世界冠军。为此泰勒就会听从教练的教导，每天进行大量的壁球练习，同时在饮食上也做了控制。教练不允许他们吃油炸食品。

泰勒从小就馋汉堡，所以他决定，当他拿到冠军的那一天，一定要买下 4 个汉堡一口吃掉。光是想着这件事，泰勒的口水直流，但他忍住了。

皇天不负有心人，泰勒参加了比赛，拿到了冠军，虽然不是世界级的，但也足以证明了他在壁球界的实力。

梦想既然实现了，那就是该是付诸承诺的时候了。于是泰勒进入了肯德基，买了四个不同口味的汉堡。你说泰勒会一口气吃掉这 4 个汉堡吗？

他没有吃掉，当这 4 个汉堡放到他面前的时候，泰勒瞬间没有了胃口。他本以为自己会很幸福的，结果并不是这样。

「如果我有一天达到某个目标，我肯定就幸福了。」

你有没有像泰勒这样，曾经做过这样的事情，或者现在还在做这样的事情。幸福这件事情真的像爬山一样，只要爬到了山顶就幸福了吗？

如果按照幸福这个维度对人进行划分，可以分为这样四个象限。

# 人生苦短，及时行乐

第一象限的人，现在快乐，未来痛苦。这种类型为「及时行乐」型。这种类型的人当下行乐，过后将是无尽的空虚。

严重的比如吸毒，每个吸毒的人都知道这事对未来没有半点好处，但依然愿意铤而走险，只顾当下贪图享乐。

生活中有很多事情也属于及时行乐的类型：刷剧、玩游戏等。我不怎么刷剧，但是我经常玩游戏。游戏这东西相当吸引人，现在普及的人群也非常广。

有时候周末一玩就是一天，然后到了晚上放下手机睡觉的时候就会觉得莫名的空虚。这说明了啥？无意义的快乐并不能让人感到幸福。

现在的躺平就是说的也是这种类型的人，目前很多 90 后认为人生这场游戏太难闯关，随便一个副本关卡（房子、车子、北京户口）都是炼狱级别的，他们连小号（孩子）都懒得练。

于是这群年轻玩家干脆不按套路出牌：什么副本任务、主线任务？老子不过关了，躺平总可以吧？及时行乐总可以吧？

# 忙碌奔波，忍辱负重

第二象限的人，现在痛苦，未来幸福。这种类型为「忙碌奔波」型。这种类型的人最常见，这种人认为现在努力为目标奋斗，等目标实现后就可以永远都幸福了，我也属于这种人。

等我以后有钱了… …

等我以后有房子了… …

等我以后还完房贷了… …

所以现在苦点不算什么。

这个是最要命的，这类型的人在当下这一刻永远也感受不到幸福。

你认为中五百万彩票的人他们感受到的幸福能持续多久？

你认为一个人突然因为车祸失去双腿的人他感受到的悲伤能持续多久？

答案是一个月。这种突如其来的幸福和从天而降的厄运只能影响一个人一个月左右的时间，一个月后，这个人的幸福水平又回到了他原本的高度。

幸福是一种体验与感受。我们缺乏的不是幸福，而是发现幸福的双眼，体验幸福的内心与走向幸福的方向。

记得曾仕强老师说过一句话：

我们总是认为是一件事把我变得不开心，其实你错了，是你对这件事的态度把你变得不开心。

# 心灰意冷，前途渺茫

第三象限的人，现在感到痛苦，认为未来也会很痛苦。这种类型为「虚无主义」型。

这种类型的人最无助，也陷的最深，他们对生活完全失去了希望。

有一个实验，三只小狗，小狗 A 和小狗 B 关到笼子里面，小狗 C 不做任何处理。

其中小狗 A 和小狗 B 的地面都铺有电线，小狗 B 的笼子边上有个开关。

实验开始，实验人员定期向地面送电。刚开始笼子里面的 A 和 B 两只小狗都很活跃，遇到电流就开始大声喊叫，同时四处打转，希望能找到出口。

不久后，小狗 B 在笼子里面找到开关，它发现这个开关可以断电。于是每次地面通电后它就去踩开关。

小狗 A 就比较惨了，笼子里面没有任何开关，它只能躺着被电（这是什么丧心病狂的实验）。

几个月后，将小狗 A 和 B 的笼子去掉，然后实验人员同时向三只小狗的地面通电，你猜结果怎样？

踩开关的小狗 B 和没有任何处理的小狗 C 触电后马上就跑掉了。没有断电开关的小狗 A 呢？它没有跑，只是躺在地面上哀嚎。

小狗 A 就是「虚无主义」的典型代表，它对未来完全失去了希望，实验人员将这种现象称为「习得性无助」。

# 感悟幸福，人生赢家

第四象限的人，认为现在很幸福，未来也会很幸福。这种类型为「感悟幸福」型。

这种类型的人最幸福，天天美滋滋。想不想成为这种类型的人呢？

有这么几种办法，这些办法当然不是赚钱，因为赚钱这事，我也不会。

第一，主动去发现生活中的幸福催化剂：美丽的环境、欢声笑语、送给爱人的一朵鲜花以及干净整洁房屋等等。

心流的作者米哈里·契克森米哈伊说过：

人类最好的时候，通常是在追求某一目标的过程中，把自身实力发挥的淋漓尽致之时。

第二，给自己设立目标，这些目标不要高的太离谱，比如你要拯救地球。也不要太简单，比如每天呼吸。要设立难度适中的目标，然后努力去实现。

每个目标的实现过程中和目标完成时，你就会感受到幸福。

第三，像个孩子一样时刻保持着永无止境的好奇心，重要的不是答案，而是那句「为什么」。

对于大多数人而言，你认为自己有多幸福，你就会有多幸福，因为思维会改变你的行动，行动改变了，人生也就改变了。
